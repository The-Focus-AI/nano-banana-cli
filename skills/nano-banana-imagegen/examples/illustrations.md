# Illustration Prompts

## Editorial Illustrations

### Concept Illustration
```
Editorial illustration representing "digital transformation" for a
business magazine. Abstract visualization of analog-to-digital transition
using geometric shapes morphing from organic to digital forms. Muted
professional colors with one accent color. Sophisticated, thought-provoking.
```

### Explainer Graphic
```
Explanatory illustration showing "how machine learning works" for a
non-technical audience. Friendly visual metaphor using everyday objects.
Flat design style, clear visual hierarchy, approachable. Pastel color
palette with good contrast.
```

## Character Illustrations

### Friendly Mascot
```
Cute mascot character for a children's educational app. Friendly owl
wearing glasses and holding a book. Soft rounded shapes, warm colors,
inviting expression. Vector illustration style, simple enough for animation.
```

### Professional Avatar
```
Professional illustration of a diverse team of business people. Modern
flat design style, inclusive representation, warm and approachable.
Clean lines, subtle shadows, cohesive color palette. Suitable for
corporate communications.
```

## Decorative Illustrations

### Pattern Background
```
Seamless pattern design featuring abstract botanical elements. Soft
sage green and cream colors. Organic shapes, hand-drawn aesthetic,
suitable for backgrounds. Tileable, subtle, elegant.
```

### Spot Illustration
```
Small decorative illustration of a coffee cup with steam curling into
whimsical shapes. Cozy, inviting, suitable as a spot illustration for
a lifestyle blog. Warm colors, hand-drawn feel.
```

## Technical Illustrations

### Diagram Style
```
Technical diagram illustration showing a network architecture. Clean
isometric style, labeled components, clear visual hierarchy. Blue and
gray professional color scheme. Informative, not decorative.
```

### Process Flow
```
Visual representation of a 5-step process. Connected icons or symbols
with arrows showing flow direction. Minimal, clean, easy to understand
at a glance. Corporate-appropriate style.
```

## Artistic Styles

### Watercolor
```
Watercolor style illustration of a cozy reading nook. Soft washes of
color, visible brushstrokes, dreamy atmosphere. Warm afternoon light,
books and plants, inviting and peaceful.
```

### Line Art
```
Elegant line art illustration of a city skyline. Single continuous line
or minimalist strokes. Black on white, sophisticated, suitable for
print. Captures recognizable landmarks with minimal detail.
```

### Retro/Vintage
```
Retro-style illustration in 1960s advertising aesthetic. Bold colors,
geometric shapes, halftone textures. Nostalgic but fresh, eye-catching,
memorable. Mid-century modern design influence.
```
