# Icon and Logo Prompts

## App Icons

### Meditation App
```
Minimalist app icon for a meditation app. Simple lotus flower symbol
centered on a soft lavender gradient background. Clean geometric design
that works at small sizes. Calming, modern, approachable. Square format.
```

### Fitness Tracker
```
App icon for a fitness tracking app. Abstract running figure or heartbeat
line in energetic orange on a dark background. Dynamic, motivating, bold.
Simple enough to be recognizable at 64x64 pixels.
```

### Notes App
```
App icon for a note-taking app. Stylized pencil or paper icon with a
folded corner. Warm yellow and white color scheme. Friendly, simple,
instantly recognizable. Flat design with subtle shadow.
```

### Weather App
```
Weather app icon featuring a stylized sun partially behind a cloud.
Cheerful blue sky gradient background. Soft, friendly, approachable.
Clean vector-style illustration.
```

## Logos

### Tech Startup
```
Modern logo mark for a tech startup. Abstract geometric shape suggesting
innovation and growth. Single color (works in any color). Clean lines,
balanced proportions, memorable silhouette. No text.
```

### Creative Agency
```
Logo symbol for a creative design agency. Artistic brush stroke or paint
splash forming an abstract shape. Energetic, creative, professional.
Works in single color and full color versions.
```

### Environmental Nonprofit
```
Logo icon for an environmental organization. Stylized leaf or tree
integrated with circular shape. Green and blue earth tones. Organic
yet professional. Simple enough for small applications.
```

## Favicons

### Generic Website Favicon
```
Simple favicon design. Single letter or abstract symbol that reads
clearly at 32x32 pixels. High contrast, bold shapes, no fine details.
Works on both light and dark backgrounds.
```
